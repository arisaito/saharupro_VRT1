## VRT#1

webVR by aframe(html, css, JavaScript)
2019~

VRT#1 は「#すこしだけ世界を味方にする服」というコンセプトの【遊べるアパレル】です。

アパレルにデザインされた QR コードをスマホで読み取ると webVR のゲームで遊べて、服を着た人と出会った人のコミュニケーションを加速させます。
